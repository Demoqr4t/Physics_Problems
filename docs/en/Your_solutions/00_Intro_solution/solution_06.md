To find the local maxima or minima of a quadratic function, we can use the **First Derivative Test** or the **Vertex Formula**. Since $f(x) = 3x^2 - 12x + 7$ is a parabola that opens upward (because the coefficient of $x^2$ is positive), we know it will have a **local minimum** at its vertex.

### 1. Find the Derivative

First, we find the derivative $f'(x)$ to determine the slope of the function:


$$f'(x) = \frac{d}{dx}(3x^2 - 12x + 7)$$

$$f'(x) = 6x - 12$$

### 2. Find the Critical Point

A local maximum or minimum occurs where the slope is zero ($f'(x) = 0$):


$$6x - 12 = 0$$

$$6x = 12$$

$$x = 2$$

---

### 3. Determine the y-coordinate

To find the value of the function at this point, substitute $x = 2$ back into the original equation:


$$f(2) = 3(2)^2 - 12(2) + 7$$

$$f(2) = 3(4) - 24 + 7$$

$$f(2) = 12 - 24 + 7$$

$$f(2) = -5$$

### 4. Classify the Point

Since the leading coefficient ($3$) is positive, the parabola opens upward, meaning the vertex is a **local minimum**.

---

### Final Answer:

The function has a **local minimum** at the point **$(2, -5)$**. There are no local maxima for this function as it continues to increase to infinity in both directions.
